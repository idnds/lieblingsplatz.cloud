## Cloud Migration

![Cloud Migration](./cloud-migration.jpg)

Es gibt eine Vielzahl an guten Gründen, um von lokaler Infrastruktur in eine Cloud-Umgebung umzuziehen.

* erhöhte Energie- und Betriebskosten
* veraltete Hardware
* gestiegene Anforderungen
* unterdimensionierte Infrastruktur
* fehlende Fachkräfte
* uvm.

So vielfältig die Gründe auch sein mögen, wir unterstützen in jedem Fall! So wird im Handumdrehen aus einer Hosted- oder OnPremise-ELO ECM Suite ein Lieblingsplatz.

### Migrationspfad

#### Erprobt

Wir bewegen uns auf einem erprobten Migrationspfad, der auch mit sehr großen Datenmengen beschritten werden kann. Bereits im laufenden Betrieb kann die Migration begonnen werden. So ist zum Umstellungstermin schon alles dort, wo es sein soll.

#### Aktuell

Beim Bezug des Lieblingsplatz wird die aktuellste [LTS Version](/erste-schritte/glossar/#lts-version) der ELO ECM Suite eingesetzt. Ausstehende Updates werden im Rahmen des Umzugs vorgenommen.

## Pläne vergleichen

### Lieblingsplatz (Plan 1) Basis

#### Infrastruktur inklusive

* Eigene Sub-Domäne unter lieblingsplatz.cloud
* Abgesichert mit TLS-Zertifikat (HTTPS) zum gesicherten und vertrauenswürdigen Zugriff.
* Datenbank & Dokumentenarchiv
  * hochverfügbar
  * redundant
  * inkl. Datensicherung (14 Tage und Tape-Auslagerung)

#### Lizenzen inklusive

* ELO ECM Suite (ELOprofessional) für 5 Named-Full-User
* ELO Web Client
* ELO Desktop Client *Plus*
* ELO Mobile Client
* ELO Java Client

* Die Umgebung für Testsysteme.
